How to convert text cases online

  1. Type or paste your text content in the Input Text Data box.
  2. Click any of the case buttons: UPPER CASE, lower case, Title Case, or Sentence case.
  3. The output capitalization updates instantly in the right-side box.
  4. Click Copy to copy the output text.

Frequently Asked Questions

What is Sentence Case conversion?

Sentence Case format capitalizes the first character of each parsed sentence sequence, converting all subsequent characters in the sentence to lowercase. This is useful for formatting copy paragraphs.

Are my texts private?

Yes. All operations run locally inside your browser memory cache. None of your text values are uploaded to our servers.
