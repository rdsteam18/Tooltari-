How to Split a PDF online

  1. Drag and drop your PDF file or click Select File to upload.
  2. In the configuration field, type the page ranges you want to extract (e.g., 1-3, 5, 8-10).
  3. Click on Split PDF Now. ToolTari segment calculations are instant.
  4. Click the individual download links in the output panel to save each file.

Frequently Asked Questions

What formatting is accepted for page ranges?

You can use commas to separate different output files and hyphens to define page spans. For example: 1-3, 5, 7-10 generates three separate PDF documents: one with pages 1, 2, and 3; one containing page 5; and one containing pages 7 through 10.

Are my files uploaded?

No. ToolTari compiles and splits your PDF document structures entirely inside your local browser memory cache. Your document contents remain completely secure.
