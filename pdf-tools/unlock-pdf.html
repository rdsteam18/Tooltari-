How to Unlock a PDF file

  1. Drag your locked PDF into the upload zone or click Select File.
  2. In the settings box, type the file's current security password.
  3. Click on Unlock PDF Now. ToolTari clears the file encryption tags.
  4. Download the stripped PDF document. It will no longer prompt you for passwords.

Frequently Asked Questions

Can this decrypt a PDF if I don't know the password?

No. To decrypt standard PDF security arrays, you must enter the valid user or owner password. ToolTari does not brute-force or bypass cryptographic locks.

Is this secure?

Yes. Because the decryption occurs 100% locally inside your browser window, your password keys never cross the network. This makes it completely safe to use for banking statements and legal agreements.
