How to Protect your PDF with a password

  1. Drag your document into the box or choose Select File.
  2. Enter a secure password in the input field.
  3. Click on Encrypt PDF Now. ToolTari wraps the file locally.
  4. Save the locked PDF. Opening it will now request the password.

Frequently Asked Questions

What encryption standard is used?

ToolTari utilizes the standard PDF document user and owner protection algorithm provided by pdf-lib. This creates a standard PDF permissions dictionary block which is recognized by Chrome, Adobe Reader, and all standard PDF decoders.

Can ToolTari recover my password if I forget it?

No. Because all computation is client-side and we do not store passwords or documents, we have absolutely no way to access or recover forgotten credentials.
